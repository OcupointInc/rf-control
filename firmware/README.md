# Firmware binaries

Drop the built `.uf2` files for each eval board here:

- `black_canyon.uf2`
- `straps.uf2`
- `whalepod.uf2`

See [docs/firmware/README.md](../docs/firmware/README.md) for the
drag-and-drop reflash procedure. The firmware source is maintained
in a separate repo; only the built binaries are tracked here.
